# Node-Mongo-CRUD

A basic CRUD functionality in Node js and Mongodb

## Requirement

Node must be installed 


## Installation

Run the following command in cmd

```bash
git clone https://github.com/rashmipriya2014/Node-Mongo-CRUD.git

cd Node-Mongo-CRUD

npm install

npm start

```

## Run on browser
http://localhost:3000/
